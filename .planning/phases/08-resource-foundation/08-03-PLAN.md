  <action>
Apply these mechanical replacement patterns across all 5 dialog files:

**Field label pattern (most common -- appears 8x in ConnectionEditor, 5x in GroupEditor, 3x in ChangePassword, etc.):**

Before:
```xml
<TextBlock Text="Name" FontSize="14"
    Foreground="{DynamicResource TextFillColorSecondaryBrush}" Margin="0,0,0,4" />
```
After:
```xml
<TextBlock Text="Name" Style="{StaticResource FieldLabelStyle}" />
```

The `FieldLabelStyle` already includes FontSize=14, Foreground=TextFillColorSecondaryBrush, and Margin=0,0,0,4. Remove ALL three inline attributes and add the Style reference.

**Form field spacing pattern:**

Before: `<ui:TextBox Text="{Binding Name}" Margin="0,0,0,12" />`
After: `<ui:TextBox Text="{Binding Name}" Margin="{StaticResource FormFieldSpacing}" />`

**Section label pattern (in dialogs with multiple sections):**

Before:
```xml
<TextBlock Text="SECURITY" FontSize="11"
    Foreground="{DynamicResource TextFillColorSecondaryBrush}" Margin="0,8,0,4" />
```
After:
```xml
<TextBlock Text="SECURITY" Style="{StaticResource SectionLabelStyle}"
    Margin="{StaticResource SectionLabelMargin}" />
```

**Error text pattern:**

Before:
```xml
<TextBlock Text="{Binding ErrorMessage}" FontSize="12"
    Foreground="{DynamicResource DeskbridgeErrorBrush}" TextWrapping="Wrap" />
```
After:
```xml
<TextBlock Text="{Binding ErrorMessage}" Style="{StaticResource ErrorTextStyle}" />
```

**Card container pattern (in dialogs that use card-like borders):**

Before:
```xml
<Border Background="{DynamicResource ControlFillColorDefaultBrush}" CornerRadius="4"
    BorderBrush="{DynamicResource ControlStrokeColorDefaultBrush}" BorderThickness="1" Padding="12">
```
After:
```xml
<Border Style="{StaticResource CardContainerStyle}">
```

**Dialog content margin (on the main content panel):**
Replace `Margin="16,12"` or similar dialog padding with `Margin="{StaticResource DialogContentMargin}"`.

**CommandPaletteDialog specifics:**
- The `Padding="6,2"` on keyboard shortcut badges is intentionally off-grid for micro-elements -- leave these as-is.
- Result row TextBlocks with caption-style text --> `Style="{StaticResource CaptionStyle}"`
- Search input styling: leave WPF-UI TextBox styles alone, only replace margin/padding inline values.

**CRITICAL rules:**
- Do NOT change element structure, bindings, commands, Visibility bindings, or behavior.
- Do NOT add implicit styles. Only add explicit `Style="{StaticResource ...}"` attributes.
- Keep `DynamicResource` for brushes, `StaticResource` for Thickness/Style.
- If a Margin/Padding does NOT match a named resource, leave it inline.
- Preserve all `x:Name`, `Grid.Row`, `Grid.Column`, and other layout attributes.
