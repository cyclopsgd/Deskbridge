Migrate MainWindow.xaml from inline style values to shared resource references. Apply these specific replacement patterns:

**Typography replacements:**
- Every `TextBlock` with `FontSize="11"` + `Foreground="{DynamicResource TextFillColorSecondaryBrush}"` pattern --> add `Style="{StaticResource SectionLabelStyle}"` and REMOVE inline FontSize and Foreground attributes. Keep any inline Margin that differs from the style default (section labels in MainWindow may have Margin="0,8,0,4" which matches `SectionLabelMargin` -- replace those with `Margin="{StaticResource SectionLabelMargin}"`).

- Every `TextBlock` with `FontSize="12"` + `Foreground="{DynamicResource TextFillColorSecondaryBrush}"` --> `Style="{StaticResource CaptionStyle}"` and remove inline FontSize + Foreground.

- Every `TextBlock` with `FontSize="14"` + `Foreground="{DynamicResource TextFillColorPrimaryBrush}"` (or no explicit Foreground) --> `Style="{StaticResource BodyStyle}"` and remove inline FontSize + Foreground.

- Every `TextBlock` with `FontSize="14"` + `Foreground="{DynamicResource TextFillColorSecondaryBrush}"` + `Margin="0,0,0,4"` --> `Style="{StaticResource FieldLabelStyle}"` and remove inline FontSize, Foreground, and Margin.

- Every `TextBlock` with `FontSize="20"` + `FontWeight="SemiBold"` --> `Style="{StaticResource SubtitleStyle}"` and remove inline FontSize, FontWeight, Foreground.

- Every `TextBlock` with `FontSize="16"` + `FontWeight="SemiBold"` --> `Style="{StaticResource CardTitleStyle}"` and remove inline FontSize, FontWeight, Foreground.

- Every `TextBlock` with `FontSize="12"` + `Foreground="{DynamicResource TextFillColorTertiaryBrush}"` --> `Style="{StaticResource HintStyle}"` and remove inline FontSize, Foreground.

**Spacing replacements:**
- `Margin="0,0,0,4"` on field labels --> `Margin="{StaticResource FormFieldLabelMargin}"`
- `Margin="0,0,0,12"` on form fields --> `Margin="{StaticResource FormFieldSpacing}"`
- `Margin="0,0,0,16"` between sections --> `Margin="{StaticResource SectionSpacing}"`
- `Margin="0,8,0,4"` on section labels --> `Margin="{StaticResource SectionLabelMargin}"`
- `Padding="12"` on card containers --> `Padding="{StaticResource CardPadding}"`
- `Padding="16"` on dialog content --> `Padding="{StaticResource CardPaddingLarge}"`
- `Margin="8,0"` on status bar content --> `Margin="{StaticResource StatusBarContentMargin}"`

**Container replacements:**
- `Border` elements with `Background="{DynamicResource ControlFillColorDefaultBrush}"` + `CornerRadius="4"` + `BorderBrush="{DynamicResource ControlStrokeColorDefaultBrush}"` + `BorderThickness="1"` + `Padding="12"` --> `Style="{StaticResource CardContainerStyle}"` and remove all those individual attributes.

- Panel areas (side panel, icon rail region) with `Background="{DynamicResource ControlFillColorDefaultBrush}"` --> replace with `Background="{DynamicResource DeskbridgePanelBackgroundBrush}"` to establish Layer 1 fill.

**CRITICAL rules:**
- Do NOT change any element structure, bindings, commands, or behavior -- only style/visual attributes.
- Do NOT change Width/Height values on layout Grid RowDefinitions/ColumnDefinitions (those are structural).
- Do NOT add implicit styles. Only add `Style="{StaticResource ...}"` as explicit attributes.
- Keep `DynamicResource` for all brush references. Use `StaticResource` for Thickness/Double/CornerRadius/Style references.
- If a Margin or Padding value does NOT match any named resource exactly, leave it inline. Only replace values that have an exact match.
- Leave `Margin="0"` and `Padding="0"` inline (no resource needed for zero).
