    4. Create `src/Deskbridge.Core/Services/MRemoteNGImporter.cs`:
       - Implements `IConnectionImporter`.
       - `SourceName` returns `"mRemoteNG"`.
       - `FileFilter` returns `"mRemoteNG Config (*.xml)|*.xml"`.
       - `ParseAsync`:
         a. Create `XmlReaderSettings` with `DtdProcessing = DtdProcessing.Prohibit` (XXE prevention per STRIDE T-07-08).
         b. Use `XDocument.Load(XmlReader.Create(stream, settings))`.
         c. Wrap in try/catch for `XmlException` — rethrow as `ImportException("The selected file is not a valid mRemoteNG configuration file.")`.
         d. Check `root.Attribute("FullFileEncryption")?.Value` — if equals `"True"` (case-sensitive per mRemoteNG), throw `ImportException("This file appears to be encrypted. Please export an unencrypted version from mRemoteNG (File > Export).")`.
         e. Iterate `root.Elements("Node")`, call recursive `ParseNode(XElement)`.
         f. Count total connections and folders recursively.
         g. Return `new ImportResult(nodes, totalConns, totalFolders)`.
       - `ParseNode(XElement)`:
         a. `Type`: `el.Attribute("Type")?.Value == "Container"` -> `ImportNodeType.Container`, else `ImportNodeType.Connection`.
         b. `Protocol`: Map via switch expression: `"RDP"` -> `Protocol.Rdp`, `"SSH2"` or `"SSH1"` -> `Protocol.Ssh`, `"VNC"` -> `Protocol.Vnc`, `_` -> `Protocol.Rdp`.
         c. `Port`: `int.TryParse(el.Attribute("Port")?.Value, out var p) ? p : 3389`.
         d. `Password` attribute is EXPLICITLY SKIPPED — do not read it, do not store it, do not log it.
         e. `Description` maps from `el.Attribute("Description")?.Value`.
         f. Recursively parse children via `el.Elements("Node")`.
       - Sanitize imported names: strip path separators (`/`, `\`) and control characters (chars < 0x20 except tab/newline).
