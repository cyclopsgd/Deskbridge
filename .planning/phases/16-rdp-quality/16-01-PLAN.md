<objective>
Implement RDP resolution matching so sessions render at native viewport resolution instead of a hardcoded 1920x1080, and add dynamic resize support so the remote session adjusts when the window is resized. This eliminates the StretchBlt blur (STAB-03) and the grey border caused by DPI rounding mismatch (STAB-04).

Purpose: RDP sessions currently request 1920x1080 and use SmartSizing=true to stretch the bitmap into whatever viewport size exists. This causes visible blur on text and UI elements. The fix measures the viewport's physical pixel dimensions, sets DesktopWidth/Height to match, uses SmartSizing=false for 1:1 pixel mapping, and dynamically adjusts resolution on resize via UpdateSessionDisplaySettings.

Output: Modified RdpConnectionConfigurator, RdpHostControl with dynamic resize, ViewportMeasurement helper, IMsRdpExtendedSettings COM interface, MainWindow viewport measurement integration, unit tests for scale factor and pixel measurement logic.
</objective>

    **4. Modify `src/Deskbridge.Protocols.Rdp/RdpHostControl.cs`**

    Add a `_loginComplete` boolean field (default false). Set it to `true` in `OnLoginComplete`. This gates dynamic resize calls (per 16-RESEARCH.md Pitfall 4 -- calling `UpdateSessionDisplaySettings` before login causes `E_FAIL`).

    Add a public method `UpdateResolution(int pixelWidth, int pixelHeight, double dpiPercent)`:
    ```csharp
    /// <summary>
    /// Dynamically updates the remote session resolution without reconnecting.
    /// Only callable after OnLoginComplete fires. Falls back to SmartSizing=true
    /// if the server does not support dynamic resolution (xrdp, RDP < 8.1).
    /// [CITED: 16-RESEARCH.md Pattern 4, Pitfall 4, Pitfall 5]
    /// </summary>
    public void UpdateResolution(int pixelWidth, int pixelHeight, double dpiPercent)
    {
        if (_disposed || _rdp is null || !_loginComplete || _dynamicResizeFailed) return;

        pixelWidth = ViewportMeasurement.ClampDesktopDimension(pixelWidth);
        pixelHeight = ViewportMeasurement.ClampDesktopDimension(pixelHeight);
        var (desktopScale, deviceScale) = ViewportMeasurement.GetScaleFactors(dpiPercent);

        try
        {
            _rdp.UpdateSessionDisplaySettings(
                (uint)pixelWidth, (uint)pixelHeight,
                (uint)pixelWidth, (uint)pixelHeight,
                0,  // orientation: landscape
                desktopScale, deviceScale);
        }
        catch (COMException ex)
        {
            // Server does not support dynamic resolution -- fall back to SmartSizing
            _dynamicResizeFailed = true;
            _rdp.AdvancedSettings9.SmartSizing = true;
            _logger.LogWarning(
                "UpdateSessionDisplaySettings not supported for {ConnectionId}: {ExceptionType} HResult=0x{HResult:X8} -- falling back to SmartSizing",
                ConnectionId, ex.GetType().Name, ex.HResult);
        }
        catch (Exception ex) when (ex is not OutOfMemoryException and not StackOverflowException)
        {
            // Unexpected error -- log and disable dynamic resize for this session
            _dynamicResizeFailed = true;
            _logger.LogWarning(
                "UpdateResolution unexpected error for {ConnectionId}: {ExceptionType} -- disabling dynamic resize",
                ConnectionId, ex.GetType().Name);
        }
    }
    ```
