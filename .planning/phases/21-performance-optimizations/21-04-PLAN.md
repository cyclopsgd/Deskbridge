    - 3.1 Cold-start Deskbridge 3 times. Each launch: confirm the main window does NOT appear empty and then populate. The window must appear with the tree ALREADY populated. PASS if zero empty-tree flashes seen across 3 launches; FAIL if any flash observed.

    - 4.1 Scroll the tree with mouse wheel. Confirm scroll advances by pixel (smooth glide), not per-row (steppy). PASS if visibly smooth; FAIL if rows snap.

    - 4.3 Verify no regression in: row hover highlight, selection highlight, group expand/collapse animation, multi-select Ctrl+click, drag-drop reorder, context menu open. PASS if all still work; FAIL if any visible regression.
