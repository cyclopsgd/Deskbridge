<objective>
PERF-02 — Add a 250ms debounce on `ConnectionTreeViewModel.SearchText` so rapid keystrokes coalesce into a single filter execution. Clearing the search bypasses the debounce and restores the tree synchronously.

Purpose: D-01/D-02/D-03 (CONTEXT.md) lock the timing contract. The pattern mirrors the resize-debounce in `MainWindow.xaml.cs:411-418`. Output: `IDebouncer` interface + production impl + VM refactor + xUnit test class + DI registration. TDD ordering: RED (tests + interface, build fails) → GREEN (production impl + VM ctor change, tests pass).
